## Node
1. Install NVM - The standard curl script works: https://github.com/nvm-sh/nvm#installation-and-update
2. Install NodeJS 12.9 - See the NVM instructions

## Expo install
Just follow - https://docs.expo.io/versions/latest/get-started/installation/
When it asks you whether to use yarn, say no. Make sure you're using npm. 

## Ant Design
This is a pain in the ass
1. Install react native CLI - https://facebook.github.io/react-native/docs/getting-started. Follow the expo instructions
2. We need to install the mobile rn version. Go to https://rn.mobile.ant.design/docs/react/introduce
3. Use the npm commands to install ant design. 

## React Navigation
Just follow https://reactnavigation.org/docs/en/getting-started.html. Use the expo instructions
